# 🧠 Sleep Health Analysis & Sleep Schedule Predictor  
### (5th Semester Mini Project – Machine Learning)

This repository contains our **5th Semester Machine Learning Mini Project**, completed as a **team project**, focused on analyzing **sleep health patterns** of B.Tech 3rd year students and building a **basic sleep schedule predictor**.

The objective of this project was to **apply machine learning concepts practically**, understand real-world data challenges, and build a simple end-to-end ML application using a web interface.

---

## 🎯 Project Objective

The main goals of this project were:

- To collect real-world sleep and lifestyle data from students
- To perform data cleaning, preprocessing, and analysis
- To apply machine learning techniques on the collected data
- To build a **basic sleep schedule predictor**
- To deploy the model using a simple **web application (Streamlit)**

This project was our **first practical attempt** at building and deploying an ML-based system.

---

## 👥 Team Project Information

- **Academic Level:** B.Tech – 5th Semester  
- **Project Type:** Mini Project (Machine Learning)  
- **Data Source:** Primary data collected by the team  
- **Participants:** B.Tech 3rd year students  
- **Sections Covered:** 3 sections out of 10 sections  

---

## 📊 Data Collection

- Data was collected manually from **three different sections** of B.Tech 3rd year students.
- The dataset includes **sleep habits, daily routines, and lifestyle-related attributes**.
- Since the data was collected from real users, it contained:
  - Missing values
  - Inconsistent entries
  - Variations in responses

This made the dataset suitable for learning **real-world ML preprocessing techniques**.

---

## 🔍 Machine Learning Workflow

The project follows a standard machine learning pipeline:

### 1️⃣ Data Cleaning & Preprocessing
- Handling missing values
- Removing inconsistent or invalid entries
- Encoding categorical features
- Preparing the dataset for model training

### 2️⃣ Exploratory Data Analysis (EDA)
- Understanding relationships between sleep patterns and lifestyle factors
- Visualizing data distributions and trends
- Identifying important features affecting sleep health

### 3️⃣ Model Building
- Applied basic machine learning algorithms
- Focused on learning how models behave on real data
- Emphasis was on **conceptual understanding**, not perfection

### 4️⃣ Model Evaluation
- Checked model performance using basic evaluation metrics
- Observed limitations due to dataset size and variability
- Understood why real ML models are rarely perfect in the first attempt

---

## 🌐 Web Application (Streamlit)

To complete the project, we built a **simple web-based interface using Streamlit**.

### Features:
- Users can input sleep and lifestyle-related details
- The model predicts a **basic sleep schedule recommendation**
- Helps demonstrate how ML models can be integrated into real applications

📌 **Note:**  
The prediction is **not perfect** and was never intended to be.  
This was our **first attempt at deploying an ML model**, focused on learning and experimentation.
